Planillas de sueldos diinf

Para calcular la planilla total de sueldos, primero debes subir un archivo con las marcas de reloj, para luego ir al apartado de datos extra, donde se podrán ingresar todos los justificativos del mes e ingresar las horas extras de los trabajadores. Una vez se haya completado el ingreso, se puede mostrar la planilla por pantalla.

Ingreso de archivo

El archivo debe seguir el formato establecido, donde la primera columna es la fecha de la marca de reloj, la segunda columna la hora de entrada o salida y la tercera columna es el rut del empleado.
